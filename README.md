# Proyecto para el tercer control del CDC de la asignatura IS

## Objetivo
Gestionar una biblioteca
## Breve descripción
Se dispone de una serie de funcionalidades asociadas a ella:
- Añadir libros de forma ordenada
- Eliminar libros de la misma.
- Devolver el listado de libros que actualmente dispone la biblioteca
- ...
  
La solución seguirá el patrón MVC. Por ello, se han creado 3 paquetes para organizar el código en función de este patrón.

Se dispone de una clase BibliotecarioTest encargada de realizar algunas pruebas unitarias a la clase Bibliotecario.

El proyecto tiene naturaleza Gradle
